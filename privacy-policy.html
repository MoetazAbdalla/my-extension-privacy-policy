Privacy Policy for Medipol Discount Calculator Extension

Effective Date: 07/10/2024

Introduction

The Medipol Discount Calculator Extension does not collect, store, or share any personal data. This extension is designed to work offline and perform calculations based on user input, without transmitting data to external servers.

Data Collection and Use

This extension does not collect or transmit any personal information or data to any third parties. All calculations are performed locally on the user's device, ensuring privacy and data security.

Google Extension Permissions

As part of ensuring user privacy and compliance with Google Chrome Web Store policies, the Medipol Discount Calculator Extension only requests the permissions strictly necessary for its operation. Below is a breakdown of permissions and their usage:

  • storage: Initially requested to store user preferences or data locally, but this permission is no longer required for the extension’s core functionality. We have removed it to minimize access to user data.
  • activeTab: This permission allowed the extension to interact with the currently active browser tab. However, as it is not needed for current features, it has been removed to limit unnecessary access.
  • scripting: Previously requested to inject scripts into pages for potential future features, but since these features are not yet implemented, this permission has been removed.

The extension only requests permissions necessary for the immediate functionality of its features, following the principle of least privilege. We do not request permissions for future-proofing or unused functionality, ensuring compliance with Google’s Program Policies.

Changes to this Privacy Policy

We may update this Privacy Policy from time to time, particularly when changes are made to the permissions required by the extension. Any updates will be reflected in this document and within the extension’s listing in the Chrome Web Store.
