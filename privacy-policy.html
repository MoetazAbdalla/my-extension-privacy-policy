Privacy Policy for Medipol Discount Calculator Extension

Effective Date: 07/10/2024

Introduction

The Medipol Discount Calculator Extension does not collect, store, or share any personal data. This extension is designed to work offline and perform calculations based on user input, without transmitting data to external servers.

Data Collection and Use

This extension does not collect or transmit any personal information or data to any third parties.

Permissions and Compliance

To comply with Google’s Program Policies, we have removed the following unused permissions:

  • Storage: This permission was used for local storage but is no longer required for the extension’s core functionality.
  • ActiveTab: This permission allowed interaction with the active browser tab, which is no longer necessary.
  • Scripting: This permission was previously requested for future features, but these features have not been implemented, so the permission has been removed.

We request only the narrowest permissions necessary to provide the extension’s features in compliance with Google's policies. No unnecessary or future-proofing permissions are requested.

Changes to this Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in this document.
